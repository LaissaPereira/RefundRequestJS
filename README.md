# Refund Request

A responsive Refund Request application built with **HTML5**, **CSS3**, and **JavaScript**.

This project was developed as part of a **Rocketseat** course to practice building interactive web interfaces and manipulating the DOM with JavaScript.

## Features

- Add new expense requests
- Display expense list
- Remove expenses
- Calculate the total reimbursement amount
- Responsive layout
- Modern and clean user interface

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

Run:

Simply open **index.html** in your browser.
